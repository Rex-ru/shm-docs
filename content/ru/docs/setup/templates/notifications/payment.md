## Описание

Создайте шаблон со следующим содержимым и привяжите его к событию `PAYMENT`:

```go
{{ IF pay.money == 0 }}
Ошибка, платеж не прошел
{{ ELSE }}
Зачислен платеж на сумму: {{ pay.money }} руб.

Ваш баланс: {{ user.balance }} руб.
{{ END }}
```
